# Discovery de Producto · Set homologado

Set pedagógico **abierto** de **MetodologIA** para conducir un **discovery de producto digital** con IA —
aunque nunca hayas liderado uno. Parte de una página en blanco o de una idea; la IA te entrevista y te guía,
una pregunta a la vez, hasta una **decisión con evidencia**, un **primer slice listo para validar** y tu
**próximo experimento**.

> Método primero, (Gen)IA después. Señal sobre ruido: enamórate del problema, no de la solución.

Homólogo del set **[Trabajar Amplificado](https://javimontano.github.io/trabajar-amplificado/)** — mismo design
system MetodologIA (navy/gold, bilingüe ES/EN, dark mode), nombres simples, sin precios, tool-agnostic.

## 🧭 Los 8 entregables del recorrido

0. **Arranque** — encuadre del problema/idea + supuestos por riesgo (leap-of-faith).
1. **Matriz Es / No es / Hace / No hace** — alcance y anti-alcance.
2. **Brief de idea** — one-pager con hipótesis de valor (provisional).
3. **Product Vision Board** — visión agnóstica + trazabilidad Necesidad → Producto → Meta.
4. **Customer Journey Map** — observado vs `[HIPÓTESIS]` + microprueba.
5. **Value Proposition Canvas** — fit problema-solución (jobs/pains/gains ↔ relievers/creators).
6. **Tablero de Supuestos + Experimentos + Primer Slice** — cobra los supuestos por riesgo.
7. **Service Blueprint** — acotado al primer slice.
8. **Business Model Canvas** — veredicto honesto: problem-solution fit vs product-market fit.

Cada prompt viene en **4 formatos** (natural · parámetros · SPEC · dupla system/user), siguiendo el método
**SPEC** y el Protocolo MetodologIA (*Interpreta › Planifica › Ejecuta*), con gates de confirmación.
